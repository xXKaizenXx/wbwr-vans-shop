#  WBWR React Native Shopify Challenge

A mini Shopify e-commerce mobile app built with **Expo**, **React Native**, **TypeScript**, and **GraphQL** using the **Shopify Storefront API**. It features a product collection and detail view, with data fetched and cached via **React Query**.

---

##  Features

- ‚úÖ Fetch products tagged with `online_stock:available` from Shopify Storefront API
- ‚úÖ Display products in a 2-column grid: image, title, and price
- ‚úÖ Product detail screen with:
  - Multiple images
  - Title, description, price
  - Available variants
- ‚úÖ Smooth navigation between screens
- ‚úÖ Loading and error states
- ‚úÖ Clean, responsive layout using StyleSheet (no UI libraries)

---

## Tech Stack

- **React Native** with Expo
- **TypeScript**
- **GraphQL**
- **React Query**
- **React Navigation**
- **Shopify Storefront API**
